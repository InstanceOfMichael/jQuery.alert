### *alerts, confirms and dialogs in* ***one.***

v1

A multipurpose plugin for alert, confirm & dialog.

Examples:

```
$('body')
    .alert('Alert!')
    .confirm('Alert you sure you want to do this?')
    .prompt('Whats your favorite Italian dish?');

$.fn.alert('Hey!');
```

Here is an example without semicolons:

```
$('body')
    .alert('Alert!')
    .confirm('Alert you sure you want to do this?')
    .prompt('Whats your favorite Italian dish?')

$.fn.alert('Hey!')
```
